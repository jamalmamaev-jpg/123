# Ski Portal Austria 🏔️

Premium travel affiliate portal for discovering and booking ski experiences in the Austrian Alps.

## Features

- **Resort Discovery**: Explore top Austrian ski resorts with detailed stats.
- **Affiliate Booking**: Integrated with Expedia, SnowTrex, and GetYourGuide.
- **QR Video Bonus**: Unique system for travelers to get a free professional video of their ski run.
- **Comparison Tools**: Side-by-side comparison of resorts and hotels.
- **Responsive Design**: Optimized for mobile and desktop with a modern alpine aesthetic.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** (Build tool)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **React Router** (Navigation)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`
